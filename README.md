# Algoritmo Genético Mejorado

## Problema

Problema del viajero (TSP)

## Descripción

Este proyecto implementa un algoritmo genético para encontrar la ruta más corta entre un conjunto de ciudades.

## Algoritmo original

Utiliza selección básica, cruza simple y mutación con probabilidad fija, lo que puede provocar convergencia lenta.

## Mejoras implementadas

* Elitismo (conserva las mejores soluciones)
* Selección por torneo (mejora la calidad de los padres)
* Mutación adaptativa (evita estancamiento)

## Resultados

El algoritmo mejorado converge más rápido y alcanza menores valores de fitness en comparación con el algoritmo original.
